    ☁️ Google Sheets — Dados Compartilhados

    Conecte uma planilha do Google Sheets para que Carol e Mônica possam lançar dados juntas de qualquer dispositivo. O dashboard importa os dados automaticamente ao sincronizar.

    📋 Passo a passo rápido
    1Abra o link da planilha modelo e clique em Arquivo → Fazer uma cópia para salvar no seu Google Drive.
    2Copie o ID da planilha — é a parte do URL entre /d/ e /edit. Ex: 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms
    3Cole o ID abaixo e clique em Conectar e Sincronizar.
    4Para lançar dados, abra a planilha e preencha diretamente. Depois clique em Sincronizar no dashboard para ver os dados atualizados.
    ✅ Planilha conectada!
